
# H1B Visa Application Analysis Dashboard

## 📊 Project Overview
This project analyzes H1B visa application data using SQL, Python, and Power BI to uncover insights about job roles, wages, employers, and approval trends.

## 🛠 Tools & Technologies
- SQL (Data loading using CMD)
- Python (Data Cleaning & Processing)
- Power BI (Dashboard Visualization)

## 📂 Workflow
1. Loaded raw dataset into SQL database via CMD
2. Extracted data using Python (SQL connection)
3. Performed data cleaning and preprocessing
4. Built interactive dashboard in Power BI

## 📈 Key Insights
- Majority applications are Certified
- Top employers: Amazon, Google, Microsoft, Cognizant, TCS
- High demand for Software Engineers & Developers
- Average wage ~ $122K
- States like California, Texas, New York dominate filings

## 📌 Dashboard Features
- Application status distribution
- Employer analysis
- Job role trends
- Wage analysis
- Geographic visualization
